## ServiceCollectionExtensions Class

Extension methods for registering PropertyMapper with dependency injection\.

```csharp
public static class ServiceCollectionExtensions
```

Inheritance [System\.Object](https://learn.microsoft.com/en-us/dotnet/api/system.object 'System\.Object') &#129106; ServiceCollectionExtensions

### Remarks
All overloads register [IPropMap](IPropMap.md 'PropertyMapper\.IPropMap') as a singleton built by [PropMapBuilder](PropMapBuilder.md 'PropertyMapper\.Configuration\.PropMapBuilder'),
so the resulting instance is \<em\>frozen\</em\>: runtime calls to `Configure` are rejected
and cross\-request configuration interference is structurally impossible\.
The concrete [PropMap](PropMap.md 'PropertyMapper\.PropMap') type is intentionally not registered — inject
[IPropMap](IPropMap.md 'PropertyMapper\.IPropMap') to keep consumers decoupled from the implementation\.
